# 复盘计划助手 - 微信小程序

一个可落地的年计划工具，帮你实现高完成率的目标管理。

## 功能特性

### 🎯 年度计划工作台
- **Step 1: KPT复盘** - 回顾去年的坚持、问题和改进
- **Step 2: SMART目标** - 用5要素原则设定可量化目标
- **Step 3: 拆解每月** - 把目标拆分成每月关键任务
- **Step 4: 月度追踪** - 可视化追踪完成进度

### 📅 每日复盘（5问）
- 今天做了什么有价值的事？
- 今天学到了什么新东西？
- 今天犯了什么错误？
- 今天有什么较大的情绪波动？
- 今天遇到了什么机会？

### 🧠 深度复盘（26问）
- 反思现状（7问）
- 未来规划（11问）
- 自我提升（8问）
- 立即行动（1问）

### 📈 数据追踪
- 连续复盘天数统计
- 任务完成率追踪
- 30天热力图
- 成就徽章系统

## 技术栈

- **框架**: Taro 4.x
- **语言**: TypeScript
- **UI**: 自定义组件 + Sass
- **存储**: 微信小程序本地存储

## 开发指南

### 安装依赖
```bash
cd miniprogram
pnpm install
```

### 开发模式
```bash
pnpm run dev:weapp
```

### 构建生产版本
```bash
pnpm run build:weapp
```

### 导入微信开发者工具

1. 打开微信开发者工具
2. 导入项目，选择 `miniprogram/dist` 目录
3. 填写你的 AppID（或使用测试号）
4. 开始预览和调试

## 项目结构

```
miniprogram/
├── config/              # Taro 配置文件
├── src/
│   ├── app.tsx          # 应用入口
│   ├── app.config.ts    # 应用配置
│   ├── app.scss         # 全局样式
│   ├── pages/           # 页面
│   │   ├── index/           # 首页
│   │   ├── annual-plan/     # 年度计划
│   │   ├── daily-review/    # 每日复盘
│   │   ├── deep-review/     # 深度复盘
│   │   └── tracking/        # 数据追踪
│   └── utils/           # 工具函数
├── dist/                # 构建输出目录

└── tsconfig.json
```

## 注意事项

1. 数据存储在本地设备，请定期备份
2. 首次使用需要填写 AppID 或使用测试号
3. 建议使用微信开发者工具进行调试
